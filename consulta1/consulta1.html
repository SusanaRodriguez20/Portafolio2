CONSULTA SOBRE PROTOCOLOS DE INTERNET

HTTP

el Protocolo de Transferencia de Hipertexto, es la base de la comunicación web, permitiendo solicitar y recibir información en la World Wide Web.

TCP/IP:

el conjunto de reglas esencial en Internet garantiza la entrega ordenada y correcta de mensajes, crucial para actividades como navegar, enviar correos y hacer videollamadas.

DNS:

el Sistema de Nombres de Dominio, convierte nombres de sitios web en direcciones IP comprensibles por computadoras, facilitando el acceso a servicios en línea.

CDN:

la Red de Distribución de Contenido, acelera la entrega de contenido en línea al utilizar servidores cercanos, aliviando la carga en servidores originales.

WWW:

la World Wide Web, permite compartir contenido a través de páginas web, utilizando el protocolo HTTP y enlaces para acceder a información en servidores globales.

FTP:

el Protocolo de Transferencia de Archivos, posibilita la transferencia de archivos entre computadoras, pero se recomienda el uso de SFTP o FTPS para mayor seguridad.

HTML:

el lenguaje de marcado estructura las páginas web con etiquetas y atributos, siendo esencial para la creación de contenido en Internet.
